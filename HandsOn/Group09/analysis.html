Dataset analysis

Entities:

  • X: Longitude component expressed as a double.
  • Y: Latitude component expressed as a double.
  • OBJECTID: Identifier given to each row of the document, starting by 1 and increasing one unit per subsecuent row.
  • IDPTOVENTA:Identifier for each selling point. It is composed by the prefix "FB_" and a six digits hexadecimal number.
  • Municipio: Municipality where the selling point is located.
  • Provincia: Province of the selling point.
  • Código_Postal: Postal code of the selling point.
  • Dirección: Name of the street where the selling point is located.
  • Latitud: Latitude component of the selling point. It is the same value as the Y component but with some decimals rounded.
  • Longitud: Longitude component of the selling point. It is the same value as the Y component but with some decimals rounded.

License analysis

Publisher:

The publisher of this dataset is the Consorcio Regional de Transportes de Madrid (CRTM) who published the data in its opendata portal.

Rightsholder:

The owner of the data is the same as the publisher, the Consorcio Regional de Transportes de Madrid (CRTM).

License:

You can download and share the data under the same type of license. Also you can create any derivative work from the data and under any license but with some restrictions:

  • It is not allowed to alter/delete the metadata regarding the updating date and license.
  • You cannot let people know or insinuate that CRTM has any relation with your derived work more than the facilitator of the data (without permision).
